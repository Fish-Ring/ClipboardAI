## 内置预设

| ID | 名称 | System Prompt |
|----|------|---------------|
| `qa` | 回答问题 | 用中文回答问题，简洁明了。输出必须为纯文本，严禁使用任何 Markdown 格式符号 |
| `translate` | 翻译中文 | 将以下内容翻译为简体中文，只输出翻译结果 |
| `polish` | 润色文本 | 润色以下文本，使其更通顺专业，保持原意，只输出润色结果 |
| `summarize` | AI 总结 | 用中文总结以下内容的核心要点，简洁明了 |

---

## 创建自定义预设

1. 双击托盘图标打开设置
2. 切换到「Prompt 预设」标签
3. 点击「添加」
4. 填写 ID、名称、System Prompt
5. 点击「保存」

---

## 预设格式

```json
{
  "id": "your-preset-id",
  "name": "预设名称",
  "systemPrompt": "你的 System Prompt 内容"
}
```

- `id`：唯一标识符，建议使用英文小写和下划线
- `name`：显示在菜单中的名称
- `systemPrompt`：发送给 AI 的系统提示词

---

## 示例

### 代码审查

```json
{
  "id": "code-review",
  "name": "代码审查",
  "systemPrompt": "审查以下代码，指出潜在问题、性能优化建议和安全隐患。用中文回答。"
}
```

### 邮件润色

```json
{
  "id": "email-polish",
  "name": "邮件润色",
  "systemPrompt": "润色以下邮件内容，使其更专业、礼貌，保持原意。输出纯文本。"
}
```

### 英文翻译

```json
{
  "id": "en-translate",
  "name": "英译中",
  "systemPrompt": "将以下内容翻译为流畅、自然的中文，只输出翻译结果。"
}
```
